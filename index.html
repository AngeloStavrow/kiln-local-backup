    kiln-local-backup

    Make local backups of repositories stored on Fog Creek's FogBugz Dev Hub, also known as Kiln.

    This is a backup script to maintain a copy of your Kiln Mercurial and Git repositories on your local system.

    Dev Hub (aka Kiln) is a hosted Mercurial and Git service from Fog Creek Software.

    This script is an easy way to back up all or just a subset of your Kiln Repositories. It uses the Kiln API to get a list of repositories, so even if you add or rename repositories, you can still get a complete backup. The script supports accessing repositories by both https and SSH.

    It works on Windows, Linux and Mac OS X.
